
Recall the linear equilibrium equations (Box~\ref{box:basic-shear}):
\begin{equation}
    \begin{aligned}
        \bm{n}' &=  \mathbf{0} \\
        \bm{m}' + \FrameBasis \times \bm{n} &= \mathbf{0} \\
        \bm{w}' - \bm{v} &= \mathbf{0}
    \end{aligned}
\end{equation}

The state determination requires solving \(n_q+n_{\rm IP}\,n_s\) nonlinear equations comprising the compatibility residual \(\residv\) and \(n_{\rm IP}\) pointwise equilibrium residuals \(\resids\):
\begin{subequations}
\begin{align}
\residv[\bm{e}] &:= \bm{v}-\int_0^L \ForceInterp^{\top}(x)\,\bm{e}(x)\,\dd\reflenx = \mathbf{0}
&(\text{\(n_q\) equations})
\label{eq:def-mf-gv}
\\
\resids(\bm{q},\bm{e};\,x_{\ell}) &:= \inters(\bm{q};\,x)-\bm{s}(\bm{e}(x_{\ell}))=\mathbf{0}
&(\text{\(n_s\) equations for each \(x_{\ell}\)})
\label{eq:def-mf-gs}
\end{align}
\label{eq:def-mf-residual}
\end{subequations}
for the basic forces \(\bm{q}\in\mathbb{R}^{n_q}\) and section deformations \(\bm{e}_{\ell}\in\mathbb{R}^{n_s}\) at each integration point \(x_{\ell}\)
where:
\begin{equation}
\inters(x) = \ForceInterp(x) \bm{q} + \bm{s}_{\mathrm{p}}(x)
\label{eq:def-interps}
\end{equation}
and the matrix $\ForceInterp(x)$ contains resultant interpolation functions that express section forces in terms of basic forces. 

An efficient algorithm for solving \eqref{eq:def-mf-residual} was put forth by \cite{taylor2003mixed}. 
This is generalized in the following section to support the consistent handling of element loading under nonlinear geometry. The developments summarized in Box~\ref{box:MF}.
