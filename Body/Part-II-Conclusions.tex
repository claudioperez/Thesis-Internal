

Part~II of this dissertation presents \BRACE{}, a real-time structural health monitoring platform that the California Department of Transportation (Caltrans) now uses as its frontline tool for post-earthquake bridge assessment. 

The platform, detailed in \cref{sec:irie}, streams accelerometer data from the field to a cloud service, runs models using \Xara{} (\cref{ch:xara}), and generates structural health reports for bridges in the California highway network. 
The proposed architecture sends sensor data over a REST API, queues analyses, and processes results into actionable health scores. 
These summaries reach Caltrans operators within seconds of an event, providing a quantitative basis for closure and inspection decisions. 


Type~I simulations on the platform are enabled by \Xara{} (\cref{ch:xara}), a C++ runtime framework for implementing interpreters that has been merged into the central \OpenSees{} repository. 
The stable, encapsulated API exposed through \Xara{} eliminates barriers to integration with modern machine learning toolchains like \Jax{} and PyTorch, which rely crucially on explicit state management. 
This is central for implementing the developments of \cref{ch:autodiff}.

\Cref{ch:autodiff} presents the compositional differentiation of finite element programs. Leveraging the architecture of \cref{ch:xara}, the procedure allows for natural interaction with machine learning toolchains like \Jax{} and PyTorch.

The developments of this work are supported through an ecosystem of open-source software tools:
\begin{itemize}
\item \Xara{} is hosted at \href{https://github.com/peer-open-source/xara}{https://github.com/peer-open-source/xara}. 
\item The visualization features of the platform are released through the Python package \Veux{}. 
This library generates three-dimensional models from \OpenSees{} simulations using the modern glTF format, bypassing the need to distribute heavyweight rendering dependencies. 
Source code for the project is hosted at \href{https://github.com/stairlab/veux}{https://github.com/stairlab/veux}.

\section{Closure}
In May 2024, Caltrans renewed the \BRACE{} platform for a second development phase, expanding its coverage to bridges across California, and work is underway to grant access to over 40 Caltrans engineers. 
The platform has processed over 150 seismic events since its initial deployment.
