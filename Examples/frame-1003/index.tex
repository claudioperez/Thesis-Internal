\subparagraph{Context}
This problem addresses a common oversight in the implementation of 
distributed loading under a corotational transformation. 
Distributed loads are most commonly used to model gravitational loads on members. 
In order to model such an effect under corotational kinematics, the loads must be applied in the global coordinate system, otherwise they will act as follower loads that rotate with the corotational basis.  

\subparagraph{Setup}
This phenomenon is illustrated by the simply supported span in \cref{fig:1003-conf} which is subjected to a uniformly distributed gravitational force. 
A similar problem was considered by \cite{desouza2000forcebased} using a corotational formulation.
