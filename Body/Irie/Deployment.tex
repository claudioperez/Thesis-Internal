\subsection{Events}

Data driven methods are currently limited by data sparsity. 
The recently released Simulated Ground Motion Database (SGMD) \citep{mccallen2025open-access} addresses this limitation by enabling preemptive analysis through simulated ground motions. 
The database currently includes simulations over an 80 \unit{\kilo\meter}$\times120$ \unit{\kilo\meter} region of the San-Francisco bay area (\cref{fig:sgmd}\footnote{This graphic was created using \emph{vega-lite} \citep{satyanarayan2017vega-lite}.})
and has been linked to \BRACE{}, enabling predictive modeling for structures lacking sensor data, and improving the ability to forecast structural responses during future seismic events. 
Additionally, SGMD data can inform real-time assessments by providing baseline models against which real-time sensor data can be compared, enhancing the accuracy of structural health evaluations. 
This integration extends the scope of \BRACE{} to a broader range of infrastructure within the transportation network, contributing to more comprehensive community resilience.

\subsection{Inventory}

Throughout the development of the platform, real-time monitoring has centered around two pilot structures:
\begin{itemize}
\item The \emph{Hayward Hwy 580-238 Interchange} (\AssetID{33-0214L}) 
\item The US 101/Painter St. Overcrossing  (\AssetID{04-0236}).
\end{itemize}
