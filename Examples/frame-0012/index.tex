This example verifies consistency with Saint~Venant's solutions, as developed in \cref{sec:timo}. 
The reference continuum solution is derived in \cref{sec:sv}. 

Fiber beam models are often verified in the linear elastic range by comparing deflections against beam theory solutions \citep[5.2.1]{saritas2006mixed} and finite element simulations of a continuum problem \citep{lecorvec2012nonlinear,gruttmann2000theory}. 
This example momentarily departs from this convention and compares directly to Saint~Venant's continuum solution as treated by \cite{iesan1986saintvenants} and summarized in \cref{sec:sv}. 
The objective is to verify that the section formulation developed in \cref{sec:trace} maintains strain-level correspondence with the continuum solution as long as the structure remains in the Saint~Venant class.
This class consists of the terminal loadings described in \cref{sec:sv} in the range where material and geometric nonlinearities can be neglected.
