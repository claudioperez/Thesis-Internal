\chapter{Primary Kinematics}\label{ch:frame-hierarchy} 
%
This chapter develops specific forms for the strain
variables \(\bm{e}\) as a function of the configuration variables
\(\chi\) for particular theories. We begin with the Special Cosserat
theory, and progressively simplify towards the linear theories.

\begin{itemize}
    \item \Gls{frame:exact}
    \item \Gls{frame:delta} This is a second order tructation of the \gls{frame:exact} model.
    \item \Gls{frame:stiff} 
    \item \Gls{frame:axial} This is the simplest model which captures flexural buckling instability under pure compression.
    \item \Gls{frame:basic}
\end{itemize}
