\subsection{Equilibrium Violations}

For any stress field satisfying \eqref{eq:saint-venant-stress} and the traction-free condition \(\StressTensor\,\bm n=\mathbf{0}\) on \(\BodyLateralSurface\),
the field equations reduce on each section to
\begin{subequations}
\label{eq:sv-equilibrium}
\begin{align}
\bm{\tau}'&=\mathbf{0} && \text{in }\Section,\\
\DivS\bm{\tau}+\AxialStress'&=0 && \text{in }\Section, \label{eq:sv-axial-equilibrium}\\
\bm{\tau}\cdot\bm\nu&=0 && \text{on }\SectionBoundary.
\end{align}
\end{subequations}


Under pure torsion, a beam with a single auxiliary warping field \(\alpha\) generates strains from \eqref{eq:linear-gl-strain-pi}:
\begin{equation}
\begin{aligned}
\hat{\bm{\varepsilon}} 
&= \alpha' \WarpTwistIesan\,\FrameBasis + \Twist \FrameBasis \times (\PlanePosition-\tilde{\bm{\SectionLocation}}) + \alpha \nabla  \WarpTwistIesan \\
&= \alpha' \WarpTwistIesan\,\FrameBasis +  \Twist \, \bigl(\nabla \WarpTwistIesan + \FrameBasis\times(\PlanePosition - \tilde{\bm{\SectionLocation}})\bigr) - (\Twist - \alpha) \nabla \WarpTwistIesan
\end{aligned}
\label{eq:torsion-strain}
\end{equation}
where the Saint Venant function \(\WarpSVP\) with regard to a point \(\bm{\pi} \in \Section\) is uniquely defined up to a constant as the Riesz vector 
\begin{equation}
    (\WarpSVP,\, \psi)_G =  \frac{1}{2} \left\{ \| \PlanePosition - \bm{\pi}\|^2,\, \psi\right\}_G %= - [\nabla \psi,\, \FrameBasis, \PlanePosition]_G =
    \qquad \forall \psi \in \mathcal{F}(\Section)
    \label{eq:def-sv-riesz}
\end{equation}
or, equivalently (by \cref{prop:armero-81}), as the solution to the standard Neumann Laplace problem \eqref{eq:warp-sv-bvp}:
\begin{equation}
\left\{
\begin{array}{rll}
\DivG\left(\left.\nabla \WarpSVP+\FrameBasis \times(\PlanePosition-\bm{\pi})\right.\right) &=0 & \text { in } \Section, \\
\nabla_{\bm{\nu}} \WarpSVP &=-\FrameBasis\times(\PlanePosition-{\bm{\pi}}) \cdot \bm{\nu} & \text { on } \PlaneBoundary, \\
\int_{\Section} \WarpSVP \volE &= 0
\end{array}
\right.
% \label{eq:warp-sv-bvp}
% \quad\text{and}\quad
% \begin{array}{ll}
% \nabla \cdot\left(n_G \nabla \WarpFA \right)=n_E \WarpTwistIesan & \text { in } \Section, \\
% \nabla_{\bm{\nu}} \WarpFA =0 & \text { along } \partial \Section,
% \end{array}
\end{equation}
where \(\DivG\) is the divergence with respect to the volume form \(\mu_G\), as defined by \eqref{eq:def-div} (Box~\ref{box:recall}).

% \end{equation*}

Under elastic isotropic material response, the stress field arising from \eqref{eq:torsion-strain} is in violation of equilibrium \eqref{eq:sv-axial-equilibrium}:
\begin{equation}
\left\{
\begin{aligned}
0 &\ne \Div\bm{\PointShearStress} + \AxialStress' = 
 \Gref \DivG(\alpha\nabla\WarpTwistIesan + \Twist\FrameBasis\times(\SectionCoordinate-\tilde{\bm{\SectionLocation}})) + E\WarpTwistIesan\alpha'' \\
0 &\ne \PointShearStress\cdot\bm{\nu} =...
 \end{aligned}
 \right.
 \label{eq:torsion-violation}
\end{equation}


The residual shear stress
$G(\Twist - \alpha)\,(\FrameBasis\times\PlanePosition)$ violates
cross-sectional equilibrium in two respects: it carries a nonzero traction
$G(\FrameBasis\times\PlanePosition)\cdot\bm{\nu}$ on~$\partial\Omega$,
and its divergence
$\nabla\cdot G(\FrameBasis\times\PlanePosition)$ does not match the
distribution $E\,\WarpTwistIesan$ required by longitudinal balance of the axial stress
$E\,\alpha''\,\WarpTwistIesan$. 
The traction must vanish identically and constitutes a
hard constraint; the divergence must be proportional to $E\,\WarpTwistIesan$, but with
an amplitude that couples to the axial problem and is not determined at the cross-sectional level. 

A gradient enrichment $\nabla\chi$ is a natural choice,
as it derives from an additional warping displacement. 
However, a single
gradient field satisfying prescribed Neumann data has its divergence fully
determined. 
Two fields are therefore needed: one to absorb the inhomogeneous
boundary data, with whatever divergence results, and a second with homogeneous
Neumann data, whose amplitude remains free to supply the required interior source. 
This motivates the ansatz
\begin{equation}
  \tilde{\StrainVector}
  = (\Twist - \alpha)\,
    \bigl(\nabla\chi_1
    + \EnhanConstFA\,\nabla\chi_2\bigr),
  \label{eq:enhan-ansatz}
\end{equation}
where $\EnhanConstFA$ is a free parameter. 
The transverse shear components of the 
strain~\eqref{eq:torsion-strain} is
\begin{equation}
  \PointShearStrain % \hat{\bm{\varepsilon}}_{\gamma}
  = \alpha\bigl(\nabla\WarpTwistIesan + \FrameBasis\times\PlanePosition\bigr)
  + (\Twist - \alpha)\,\FrameBasis\times\PlanePosition.
  \label{eq:shear-decomp}
\end{equation}
The Saint-Venant warping function $\WarpTwistIesan$ is defined so that the shear pattern
$G(\nabla\WarpTwistIesan + \FrameBasis\times\PlanePosition)$ is divergence-free with
vanishing boundary traction:
\begin{equation}
  \nabla\cdot G\bigl(\nabla\WarpTwistIesan
    + \FrameBasis\times\PlanePosition\bigr) = 0
  \;\;\text{in }\Omega, \qquad
  G\bigl(\nabla\WarpTwistIesan
    + \FrameBasis\times\PlanePosition\bigr)\cdot\bm{\nu} = 0
  \;\;\text{on }\partial\Omega,
  \label{eq:sv-bvp}
\end{equation}
so the first term in~\eqref{eq:shear-decomp} is self-equilibrated for any value
of~$\alpha$. 
The residual
$(\Twist - \alpha)\,\FrameBasis\times\PlanePosition$ produces two violations of
cross-sectional equilibrium. On the boundary it generates a spurious traction
\begin{equation}
  G\,(\Twist - \alpha)\,
  (\FrameBasis\times\PlanePosition)\cdot\bm{\nu}
  \quad\text{on }\partial\Omega,
  \label{eq:torsion-violation}
\end{equation}
and in the interior, the shear stress divergence
\begin{equation}
  \nabla\cdot(G\,\hat{\bm{\varepsilon}}_{\gamma})
  = (\Twist - \alpha)\,
    \nabla\cdot G\,(\FrameBasis\times\PlanePosition)
  \label{eq:divergence-violation}
\end{equation}
fails to balance the axial stress gradient $E\,\alpha''\,\WarpTwistIesan$ required by
longitudinal equilibrium. 
We seek to address both defects by enriching the
transverse strain with a superposition of cross-sectional gradient fields:
\begin{equation}
  \bar{\StrainVector}
  = (\Twist - \alpha)\,
    \bigl(\nabla\chi_1 + \EnhanConstFA\,\nabla\chi_2\bigr).
  % \label{eq:enhan-ansatz}
\end{equation}

\paragraph{Traction.}
Choose $\chi_1$ so that
$\FrameBasis\times\PlanePosition + \nabla\chi_1$ is both traction-free and
divergence-free:
\begin{equation}
  \nabla\cdot G\bigl(\nabla\chi_1
    + \FrameBasis\times\PlanePosition\bigr) = 0
  \;\;\text{in }\Omega, \qquad
  G\bigl(\nabla\chi_1
    + \FrameBasis\times\PlanePosition\bigr)\cdot\bm{\nu} = 0
  \;\;\text{on }\partial\Omega.
  \label{eq:chi1-bvp}
\end{equation}
This coincides with~\eqref{eq:sv-bvp}, giving $\chi_1 = \WarpTwistIesan$ uniquely (up to
an additive constant). 
After this enrichment, the total shear reduces to
$\Twist(\nabla\WarpTwistIesan + \FrameBasis\times\PlanePosition)$, which is traction-free
and divergence-free. 
This eliminates~\eqref{eq:torsion-violation}
and~\eqref{eq:divergence-violation} simultaneously, but the divergence is now
zero, whereas longitudinal equilibrium demands a nonzero shear stress
divergence proportional to~$E\,\WarpTwistIesan$ to balance axial stress gradients from
nonuniform warping.

\paragraph{Divergence.}
To supply the missing divergence without disturbing the traction-free condition, $\chi_2$ must satisfy
\begin{equation}
  \nabla\cdot(G\,\nabla\chi_2) + E\,\WarpTwistIesan = 0
  \;\;\text{in }\Omega, \qquad
  G\,\nabla_{\bm{\nu}}\chi_2= 0
  \;\;\text{on }\partial\Omega,
  \label{eq:fa-bvp}
\end{equation}
whose solvability is ensured by the normalization
$\int_{\Section} \,\WarpTwistIesan \volE = 0$. 
The homogeneous Neumann condition preserves the traction-free state established by~$\WarpTwistIesan$ and the divergence $-E\,\WarpTwistIesan$ balances the axial stress gradient $E\,\alpha''\,\WarpTwistIesan$.
